## <a name="search-syntax"></a><span data-ttu-id="9864d-179">Wyszukiwanie składni</span><span class="sxs-lookup"><span data-stu-id="9864d-179">Search Syntax</span></span>

<span data-ttu-id="9864d-180">Wyszukaj pakiet NuGet działa tak samo nuget.org, z poziomu interfejsu wiersza polecenia NuGet, a w ramach rozszerzenia Menedżera pakietów NuGet w programie Visual Studio.</span><span class="sxs-lookup"><span data-stu-id="9864d-180">NuGet package search works the same on nuget.org, from the NuGet CLI, and within the NuGet Package Manager extension in Visual Studio.</span></span> <span data-ttu-id="9864d-181">Ogólnie rzecz biorąc wyszukiwanie jest stosowany do słów kluczowych, a także opisy pakietu.</span><span class="sxs-lookup"><span data-stu-id="9864d-181">In general, search is applied to keywords as well as package descriptions.</span></span>

- <span data-ttu-id="9864d-182">**Słowa kluczowe**: wyszukiwania wyszukuje odpowiednie pakiety, które zawierają wszystkie podane słowa kluczowe.</span><span class="sxs-lookup"><span data-stu-id="9864d-182">**Keywords**: Search looks for relevant packages that contain all the provided keywords.</span></span> <span data-ttu-id="9864d-183">Przykład:</span><span class="sxs-lookup"><span data-stu-id="9864d-183">Example:</span></span>

    ```
    modern UI javascript
    ```

- <span data-ttu-id="9864d-184">**Wyrażenia**: wprowadzania terminów w cudzysłowie wyszukuje dokładne dopasowania bez uwzględniania wielkości liter w tych warunkach.</span><span class="sxs-lookup"><span data-stu-id="9864d-184">**Phrases**: Entering terms within quotation marks looks for exact case-insensitive matches to those terms.</span></span> <span data-ttu-id="9864d-185">Przykład:</span><span class="sxs-lookup"><span data-stu-id="9864d-185">Example:</span></span>

    ```
    "modern UI" package
    ```

- <span data-ttu-id="9864d-186">**Filtrowanie**: wyszukiwanego terminu może dotyczyć określoną właściwość przy użyciu składni `<property>:<term>` gdzie `<property>` (bez uwzględniania wielkości liter) może być `id`, `packageid`, `version`, `title`, `tags`, `author`, `description`, `summary`, i `owner`.</span><span class="sxs-lookup"><span data-stu-id="9864d-186">**Filtering**: You can apply a search term to a specific property by using the syntax `<property>:<term>` where `<property>` (case-insensitive) can be `id`, `packageid`, `version`, `title`, `tags`, `author`, `description`, `summary`, and `owner`.</span></span> <span data-ttu-id="9864d-187">Warunki mogą być zawarte w cudzysłowie, jeśli to konieczne, i można wyszukiwać wiele właściwości, w tym samym czasie.</span><span class="sxs-lookup"><span data-stu-id="9864d-187">Terms can be contained in quotes if needed, and you can search for multiple properties at the same time.</span></span> <span data-ttu-id="9864d-188">Ponadto wyszukuje `id` właściwości są dopasowania podciąg, podczas gdy `packageid` używa dokładnego dopasowania.</span><span class="sxs-lookup"><span data-stu-id="9864d-188">Also, searches on the `id` property are substring matches, whereas `packageid` uses an exact match.</span></span> <span data-ttu-id="9864d-189">Przykłady:</span><span class="sxs-lookup"><span data-stu-id="9864d-189">Examples:</span></span>
